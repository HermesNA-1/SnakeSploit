Install

Get Started in Seconds

No dependencies, no databases, no configuration. Just Python and a terminal.

# Clone and install in 5 seconds
git clone https://github.com/HermesNA-1/SnakeSploit.git
cd SnakeSploit
python3 install.py
snakesploit
# Manual setup
git clone https://github.com/HermesNA-1/SnakeSploit.git
cd SnakeSploit
chmod +x snakesploit.py
ln -s $(pwd)/snakesploit.py ~/.local/bin/snakesploit
mkdir -p ~/.snakesploit/{cve_cache,poc_cache,loot,logs}
snakesploit
# Start the web dashboard
pip install flask
snakesploit --gui

# Open in browser: http://localhost:5000
# Or use GitHub Pages: https://hermesna-1.github.io/SnakeSploit/
